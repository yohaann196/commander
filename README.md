# Commander

Commander is a text-based RPG that runs entirely in your terminal.

## Overview
You play as the last surviving Commander in a war-torn world. 
Make choices, fight enemies, level up, and shape the outcome of the campaign.

## Features
- Turn-based combat
- Branching story decisions
- Level progression
- Random encounters

## How to Run

cd commanderv1  
python3 commander.py
